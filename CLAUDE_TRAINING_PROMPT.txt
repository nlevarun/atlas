- **Evidence-First Architecture**: Every claim must have source URL, timestamp, verbatim excerpt

- **Groq as Optional Upgrade**: Free tier (6000 requests/day), still $0.00
- **Mock Data Fallbacks**: All agents work without API keys
- **Real-Time Streaming**: WebSocket updates as agents work
