# Atlas Evaluation Harness

**Phase 6: Systematic evaluation of agent performance**

## Overview

The evaluation harness measures:
1. Evidence quality and completeness
2. Source attribution accuracy
3. Synthesis coherence
4. Prediction accuracy (Brier scores)
